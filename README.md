# webapp

Author:  `Raushan Kumar Jha`
TA: `Virendra`
- Web Application with Create, Read and Update functionality to add, read and update user through different API
   1. `POST` -  /v1/user
   2. `GET` - /v1/user/:id
   3. `PUT` - /v1/user/:id

- Health endpoint
   1. `GET` -  /healthz


Pre-requisites:
- Install Node.js
- Install mysql server 

Build and Deploy:
- Clone the repository

- Build and run:
    - cd webapp
    - npm install
    - node app.js

Testing:
 - npm test
